# Define
- Hebrew: qadash (קָדַשׁ)  
	- to consecrate, sanctify, set apart, make holy  
- Greek: hagiazo (ἁγιάζω)  
	- to sanctify, make holy, consecrate  

# Related words
- qadosh (קָדוֹשׁ) — holy, set apart  
- qodesh (קֹדֶשׁ) — holiness, sanctuary, holy thing  

# Additional Info
- Leviticus 11:44 — “ye shall therefore sanctify yourselves” (hitqaddashu – הִתְקַדְּשׁוּ)  
- Leviticus 20:7 — “Sanctify yourselves therefore, and be ye holy” (hitqaddeshu – הִתְקַדְּשׁוּ)
