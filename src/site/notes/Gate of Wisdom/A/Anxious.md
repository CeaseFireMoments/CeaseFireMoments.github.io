# Define
- Hebrew: da'ag (דָּאַג) - to be anxious, to worry, to fear; deagah (דְּאָגָה) - anxiety, worry, care 

- Greek: merimnaō (μεριμνάω) - to be anxious, to worry, to care for, to be troubled with cares 

# Related words
- merimna (μέριμνα) - anxiety, care, worry (noun form)

# Additional Info
- da'agah (דְּאָגָה) - anxiety, anxious care (OT noun)
